# Aarogya 🩺 | Seamless Doctor Appointments

**Aarogya (meaning 'health' in Sanskrit) is a modern, user-centric mobile application designed to bridge the gap between patients and doctors. Find any doctor near you, book an appointment instantly, and always stay connected with the healthcare providers you trust.**

**We are going to change the UI altogether, the images shown in this readme file is a prototype version, backend remains the same (firebase).**

---

## 🎯 About The Project

In today's fast-paced world, scheduling a doctor's visit can be a cumbersome process. Aarogya solves this by providing a simple, intuitive platform to handle all your appointment needs. Our core philosophy is to empower users, allowing them to not only discover new healthcare professionals but also to easily re-book appointments with their favorite, trusted doctors without any hassle.

### ✨ Key Features

* **Comprehensive Doctor Search:**
    * Find doctors from a wide range of clinics and hospitals near you using location-based search.
    * Filter results by specialty, clinic name, or doctor's name.

* **⭐ Your Favorite Doctors:**
    * Build a personalized list of your most trusted doctors.
    * View their availability and book an appointment with them in seconds, skipping the general search.

* **📅 Intuitive Booking System:**
    * View real-time availability of doctors.
    * Select a convenient date and time slot.
    * Receive instant confirmation of your booking.

* **🔔 Smart Notifications & Reminders:**
    * Get timely reminders for your upcoming appointments.
    * Receive notifications about confirmations, cancellations, or schedule changes.

* **📂 Appointment Management:**
    * A clean dashboard to view and manage all your upcoming and past appointments.
    * Option to easily cancel or reschedule.

## 🛠️ Built With

This project is built using modern technologies to ensure a robust and scalable application.

* **Flutter:** For building a beautiful and natively compiled application for mobile from a single codebase.
* **Firebase:** Serving as the comprehensive backend platform for:
    * **Cloud Firestore:** For a flexible, scalable NoSQL cloud database to store and sync data.
    * **Firebase Authentication:** For handling user authentication securely.
    * **Firebase Cloud Storage:** For storing user-generated content like profile pictures.

## 🚀 Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

* Flutter SDK: [https://flutter.dev/docs/get-started/install](https://flutter.dev/docs/get-started/install)
* An IDE like Android Studio or VS Code with the Flutter plugin.

### Installation

1.  Clone the repo
    ```sh
    git clone [https://github.com/PuzzleHeaded-Look852/Aarogya.git](https://github.com/PuzzleHeaded-Look852/Aarogya.git)
    ```
2.  Navigate to the project directory
    ```sh
    cd Aarogya
    ```
3.  Install dependencies
    ```sh
    flutter pub get
    ```
4.  Run the app
    ```sh
    flutter run
    ```
